# Bulk Selection & Enhanced Visual Guide

## 🎯 New Features Added

### 1. Bulk School Selection
- ✅ Checkbox in header to select/deselect all schools
- ✅ Individual checkboxes for each school
- ✅ "Select All Schools" / "Deselect All" button

### 2. Row-wise Selection
- ✅ "→" button on each row to select entire row
- ✅ Selects all available cells in that row (Physical or Career)
- ✅ Skips already booked cells

### 3. Column-wise Selection
- ✅ "↓" button above each column
- ✅ Selects all available cells in that column across all schools
- ✅ Works for both Physical and Career sections

### 4. Section-wide Selection
- ✅ "All" button for Physical Sessions (selects all P1-P9 for all schools)
- ✅ "All" button for Career Fairs (selects all CF1-CF20 for all schools)

### 5. Enhanced Visual Distinction

**Available Cells (Darker):**
- 🟢 Green background (#10B981)
- White text showing slot number (P1, CF5, etc.)
- Shadow effect
- Hover: Darker green + scale up

**Booked Cells (Faded):**
- 🔴 Light red background (#FECACA)
- 40% opacity
- No hover effect
- Disabled cursor

**Selected Cells:**
- 🔵 Blue background (#3B82F6)
- White checkmark (✓)
- Medium shadow

**Confirmed Bookings:**
- 🔵 Darker blue (#2563EB)
- White CheckCircle icon
- Large shadow
- Scale up effect

### 6. Clear Section Separation

**Physical Sessions:**
- Blue header background (#DBEAFE)
- Blue column backgrounds (#EFF6FF)
- Thick blue border on right (2px)
- "PHYSICAL SESSIONS" label with building icon

**Career Fairs:**
- Purple header background (#F3E8FF)
- Purple column backgrounds (#FAF5FF)
- Thick purple border on left (2px)
- "CAREER FAIRS" label with building icon

## 📋 How to Use Bulk Selection

### Select All Schools
```
1. Click checkbox in header (top-left)
   OR
   Click "Select All Schools" button
2. All school checkboxes get checked
3. Now you can bulk select sessions for all schools
```

### Select Entire Row
```
1. Click "→" button next to any school
2. All available cells in that row get selected
3. Works for both Physical and Career sections
```

### Select Entire Column
```
1. Click "↓" button above any column (P1, CF5, etc.)
2. All available cells in that column get selected
3. Selects across all visible schools
```

### Select All Physical Sessions
```
1. Click "All" button in Physical section header
2. All available P1-P9 cells for all schools get selected
3. Hundreds of cells selected in one click!
```

### Select All Career Fairs
```
1. Click "All" button in Career section header
2. All available CF1-CF20 cells for all schools get selected
3. Maximum bulk selection!
```

## 🚀 Quick Workflows

### Workflow 1: Book All Physical Sessions
```
1. Click "All" button in Physical section
2. All green P1-P9 cells selected
3. Click "Confirm Booking (X)"
4. Done! All physical sessions booked
```

### Workflow 2: Book Specific Column for All Schools
```
1. Click "↓" above P3 column
2. P3 selected for all 8 schools
3. Click "Confirm Booking (8)"
4. All schools get P3 booked
```

### Workflow 3: Book Entire Row for One School
```
1. Click "→" next to "DPS Delhi"
2. All P1-P9 and CF1-CF20 for DPS selected
3. Click "Confirm Booking (29)"
4. DPS fully booked
```

### Workflow 4: Mixed Selection
```
1. Click "↓" above P5 (selects P5 for all schools)
2. Click "↓" above CF10 (selects CF10 for all schools)
3. Click "→" next to Ryan (adds all Ryan slots)
4. Click "Confirm Booking (X)"
5. Complex booking done in seconds!
```

## 💡 Pro Tips

1. **Visual Scanning**: Green = available (darker), Red = booked (faded)
2. **Bulk First**: Use "All" buttons for maximum speed
3. **Column Selection**: Best for booking same session across schools
4. **Row Selection**: Best for booking all sessions for one school
5. **School Checkboxes**: Track which schools you're focusing on
6. **Clear Separation**: Blue = Physical, Purple = Career (easy to distinguish)
